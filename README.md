## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely in the browser

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yahwe388-droid/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   - Simply open `kb.html` in your web browser
   - Or serve locally using any static server

### Local Development Server (Optional)

For a better development experience, you can use a local server:

**Using Python (if installed):**
```bash
python -m http.server 8000
# Then visit http://localhost:8000/kb.html
```

**Using Node.js (if installed):**
```bash
npx serve .
# Then visit the provided localhost URL
```

## 🔧 Customization

### Colors
Modify the CSS custom properties in `:root` to change the theme:

```css
:root {
    --neon-blue: #45f3ff;    /* Primary accent */
    --neon-purple: #b57eed;  /* Secondary accent */
    --bg-color: #0d111a;     /* Background */
}
```

### Content
Update personal information in `kb.html`:
- Change name, description, and contact details
- Modify skills and project information
- Update social media links

### Animations
Adjust timing and effects in `script.js`:
- Typing animation speed
- Scroll reveal thresholds
- Parallax sensitivity
